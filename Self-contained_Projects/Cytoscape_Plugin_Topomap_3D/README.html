README File for Topomap3D Version 0.1

What is Topomap3D?

Topomap3D is a Cytoscape Plug-in that allows you to visualize a 2D network as a 3D topographical map, where areas of higher node density make up mountains in the graph, and areas with no nodes result in valleys.
Topomap3D is currently a very, very basic 3D graph visualizer. It doesn't have any frills, and there is no way to edit or select anything in the Topomap3D view.

Why would this be useful?

There are many options for data visualization that can make use of the three-dimensional layout. The three-dimensional layout also adds new potential possibilities for more intuitive superposition of multiple graphs and multiple data sets. Superposition hasn't been implemented yet, however, so right now this view is more of a proof of concept or technical demo than anything else. There is also the possibility of making use of three spatial dimensions for graph layout.

How do I run Topomap3D?

  1. Put the "Topomap3DPlugin.jar" file into the Cytoscape plugins directory (the plugins folder is located in the same place as the Cytoscape application).
  2. Launch Cytoscape
  3. Create a graph, and a view for that graph, using the standard Cytoscape tools
  4. You should see new options in the Cytoscape Plug-ins menu. One of them is "Topomap3D." Once you have created a graph and a graph view, select this menu item to get a topographical map of your graph view.

How do I change the graph once it is displayed in 3D?

There are not currently any options for editing the graph once it has been displayed in 3D. You will need to delete the current graph view (which can be accomplished by right-clicking on the network information displayed in the leftmost Cytoscape tab) and create a new view. You can then lay out this view again and run Topomap3D again.

Does Topomap3D accept three-dimensional data points?

Topomap3D currently only accepts 2D graphs. The third dimension is calculated based on the proximity of graph nodes, not from any inherent properties of the nodes.
